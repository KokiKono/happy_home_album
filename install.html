Install Manual インストールマニュアル

基本的にブラウザ(Chrome)からの操作、DBのdumpファイルを実行しておく。

１、リポジトリをクローン

下記、リポジトリをcloneしてきて、READMEのset upを実行してください。

Github リポジトリ
２、プロジェクト配下のconfig.jsonの設定

production.urlを現在のローカルIPアドレスに変更

api-keyをMicrosoft AzureでAPIキーを取得してきて設定

３、サーバー起動(make production-run)

基本的に動くはず、ご自分の実行環境に合わせて適宜変更してください。

make installを忘れずに！

４、https://your_ip:5000/management/portal/にアクセス

ここが、基本的な操作画面（ポータル画面）。画面の次へ的なボタンを押せばOK

５、iOSアプリのインストール

プロジェクトを開いてインストール

６、アプリのネット設定

サーバーにしているPCと同一ネットワークにして、IP設定。家族作成後、ログインに家族リストが出ます。

７、メインとサブ画面を開く

http://your_ip:8080/main/

http://your_ip:8080/sub/

８、あとは、ポータル画面で操作

提案処理とかするとアプリで通知が行くかも？感情により異なります。

データベースにデータを入れておく必要があります。
